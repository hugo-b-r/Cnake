# Cnake
Cnake is a simple Snake game made in C. It will work in Linux and Windows, maybe MacOS. I'll do it on my free time.

### Installation

Clone repo:
```shell
git clone https://github.com/lemoustachu/Cnake.git
```

Change directory:
```shell
cd Cnake
```

and then:
```bash
make clean
make
```
